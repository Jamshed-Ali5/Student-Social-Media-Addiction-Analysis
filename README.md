# 📱 Student Social Media Addiction Analysis

An Exploratory Data Analysis (EDA) project that investigates students' social media usage and its relationship with addiction scores, mental health, sleep habits, and demographic factors.

## 📖 Overview

This project uses Python data analysis libraries to explore a dataset containing information about students' social media behavior. The analysis aims to identify trends, patterns, and correlations between daily social media usage and various lifestyle and mental health indicators.

---

## 📊 Dataset

The dataset contains information such as:

- Student ID
- Age
- Gender
- Relationship Status
- Average Daily Social Media Usage (Hours)
- Sleep Hours per Night
- Mental Health Score
- Addiction Score

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Analysis Performed

The notebook includes:

- Data Loading
- Missing Value Check
- Duplicate Check
- Data Information & Summary Statistics
- Relationship Status Distribution
- Age Distribution
- Daily Usage vs Mental Health Analysis
- Sleep Hours vs Addiction Score Analysis
- Correlation Heatmap

---

## 📷 Visualizations

The project contains several visualizations, including:

- Count Plot
- Pie Chart
- Histogram
- Scatter Plots
- Correlation Heatmap

These visualizations help understand the relationships between social media usage and student well-being.

---

## 🎯 Key Insights

- Explore social media usage patterns among students.
- Understand the relationship between daily usage and mental health.
- Analyze how sleep duration relates to addiction scores.
- Identify important correlations between numerical variables.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Jamshed-Ali5/student-social-media-addiction-analysis.git
```

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Run the notebook

```bash
jupyter notebook
```

Open the notebook and run all cells.
